## Release Requirements

The release is not publishable until these external gates have redacted
evidence:

- npm publish dry run, provenance, and signed artifact checksums;
- live Telegram sandbox link, verification, notification, and remote action;
- live email sandbox link, verification, notification, and remote action;
- GitHub sandbox repository inspection, reviewed PR creation, and manual merge
  policy evidence;
- dependency audit and container scan with no unresolved high finding; and
- two clean release-gate runs from fresh environments.

Local source and deterministic tests may mark implementation chunks complete,
but they do not replace provider or registry evidence.
